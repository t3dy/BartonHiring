# Hiring Movers — Complete Action Mapping for Barton Springs Moving

> **Live Site:** [https://t3dy.github.io/BartonHiring/](https://t3dy.github.io/BartonHiring/)
> **Claude Code Lessons:** [https://t3dy.github.io/BartonHiring/lessons/](https://t3dy.github.io/BartonHiring/lessons/)

A comprehensive single-page guide mapping every action in the mover hiring lifecycle — from planning and budgeting through recruiting, screening, onboarding, retention, and offboarding. Each action is paired with software solutions, advertising methods, and cost estimates tailored to a 2-3 crew Austin moving company.

## What's Inside

| Section | What It Covers |
|---------|----------------|
| **Pipeline Overview** | 8-stage visual pipeline with action counts and quick navigation |
| **Plan & Budget** | Staffing needs, pay rates, role definitions, hiring budget, seasonality |
| **Recruit & Source** | Job boards, social media, referral programs, local orgs, staffing agencies |
| **Screen Applicants** | Applications, phone screens, background checks, MVR, drug tests, references |
| **Interview** | Working interviews, scenario questions, fitness assessment, offers |
| **Hire & Paperwork** | Employment forms, payroll, insurance, equipment, system access |
| **Onboard & Train** | Moving fundamentals, customer service, safety, ride-alongs, probation |
| **Retain & Grow** | Pay reviews, bonuses, career paths, culture, check-ins, benefits |
| **Manage & Offboard** | Performance tracking, discipline, no-shows, layoffs, termination, exit interviews |
| **Software Summary** | All tools organized by price tier (Free → Budget → Pro → Custom) |
| **Advertising Guide** | Recruitment channels with cost and effectiveness ratings |
| **Cost Calculator** | Interactive tool to estimate total hiring cost based on your methods |

## Key Numbers

- **51 hiring actions** mapped across 8 pipeline stages
- **30+ software tools** reviewed with pricing
- **15+ advertising methods** with effectiveness ratings
- Interactive cost calculator for budgeting

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- Lucide React (icons)
- All data is inline — no backend required
